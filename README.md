# ChatBot-IA
Ce projet est la création d'un chatbot spécialisé dans les concepts d'Intelligence Artificielle (IA). Il est conçu pour répondre aux questions factuelles et contextuelles sur un corpus de connaissances structuré autour de l'IA, de l'apprentissage automatique (Machine Learning), des réseaux de neurones, et des grands modèles de langage (LLM).
